Tips: 
+ 学代码只看书没有意义，得写代码，多练习
    JavaScript写前端逻辑
    NodeJS写后端逻辑, NodeJS与JavaScript一脉相承(Java和python也可以写后端)  
    而后端负责管理数据库
+ 语言最好先选一样学精，再学另一样
    JavaScript每年都会有新feature, 最新的feature自学（可以关注个公众号）, 我们这里只讲基础的

不太合适的理解:
+ JS放在浏览器里运行, 就作为前端语言.
+ JS放在Node里运行, 就作为后端语言(node.js). -(sudo apt install nodejs)
